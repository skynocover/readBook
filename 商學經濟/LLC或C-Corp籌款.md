# LLC或C-Corp籌款

> https://alejandrocremades.com/llc-or-c-corp-for-fundraising/

LLC或C Corp籌款——對於企業家來說，什麼是更好的選擇？  
創業可能是一項艱鉅的任務，在創業時必須考慮不同的方面。  
初創公司的所有者有很多想法，而籌款是他們最關心的問題之一。  
您註冊創業公司的實體類型會對您的企業如何籌集資金產生重大影響。

通常，您可以將創業公司註冊為兩種主要類型的實體。  
一個是 LLC，另一個是 C Corp，在法律和稅務機關 (IRS) 的眼中，兩者都是不同的實體。  
如果你想創辦一家初創公司，你必須非常仔細地選擇你想要形成的實體類型。

正如我與 Inner Circle 的成員分享的那樣，我們從頭到尾為與籌款相關的一切提供幫助 
如果您組建 LLC 或 C Corp 進行籌款，您可能必須使用不同的策略，更不用說這兩種業務結構都有在運營方面有各自的優缺點。  
由於從一種業務結構轉換為另一種業務結構可能具有挑戰性，因此最好在一開始就選擇正確的結構，這樣您就可以輕鬆獲得資金。
 
如果您正試圖根據您的業務結構在 LLC 和 C Corp 之間進行選擇，並希望確保從籌款的角度選擇正確的選項，請繼續閱讀。

本文將解釋這兩種業務結構的基本原理，並解釋 LLC 和 C Corp 的籌款方式有何不同。在本文結束時，您應該能夠知道哪種業務結構更適合您。


## LLC

LLC代表有限責任公司，這些類型的公司限制了其所有者的責任。  
如果針對 LLC 提起任何訴訟或索賠，則法院不能針對所有者的資產來解決訴訟。  
在有限責任公司的情況下，所有者的資產與其業務分開。  
因此，簡而言之，有限責任公司為其所有者提供保護，使其免受與業務相關的責任。  
但是，責任保護可能並非在所有情況下都適用，並且在某些情況下所有者可能要承擔責任。

有限責任公司在擁有單一所有者或合夥企業的小企業中最受歡迎，因為它們更容易組建，而且稅收也更簡單。  
有限責任公司產生的所有利潤都報告為所有者的個人所得稅申報表，因此有限責任公司不涉及雙重徵稅。

由於有限責任公司沒有股東，他們不能發行股票，所有者可以持有公司一定比例的所有權。
 
## C Corp

C Corp本質上是一家由眾多股東擁有的公司。  
每個股東都持有股票，代表公司股東的所有權。  
由於 C Corp 可以發行股票，因此它在籌資方面比 LLC 具有優勢。

C Corp 是大多數初創公司的理想業務結構，類似於 LLC，它保護所有者免於對業務承擔責任。  
話雖如此，與有限責任公司相比，C Corp 需要更多的文書工作和治理，並且具有更嚴格的法律要求。

當一家企業將自己註冊為 C 公司時，它必須繳納公司稅，也稱為雙重徵稅。  
這意味著成員必須在公司層面和個人層面繳納稅款。  
如前所述，有限責任公司不必繳納雙重稅，營業收入作為所有者的個人收入徵稅。

## 在 LLC 和 C Corp 之間選擇作為您的業務結構

好的，現在您知道 LLC 和 C Corp 是什麼以及它們之間的區別，您仍然不足以為您的公司選擇任何一種業務結構。  
當然，籌款是組建企業時的重要考慮因素，我們將在文章的下一節中詳細討論。  
但在我們開始工作之前，當您在有限責任公司或 C 公司之間進行籌款選擇時，還有一些額外的關鍵點需要考慮：

### 控制程度

當您以單一所有者的身份組建企業時，目標是保持對運營的控制。  
由於有限責任公司不需要董事會來進行日常運營，因此所有者可以將關鍵決策留給自己。  
此外，有限責任公司每個成員的角色在運營協議中都有明確規定。  
因此，即使有限責任公司有多個成員，創始人也可以將關鍵決策留給自己。

另一方面，C Corp 有一個董事會，負責管理公司的各種事務，包括與業務相關的關鍵決策。  
因此，C Corp 的控制級別由公司的許多股東共享。
 
當您在 LLC 或 C Corp 之間進行籌款選擇時，您應該在選擇之前考慮您想要對您的業務進行的控制級別。

### 複雜程度

您為公司選擇的業務結構也會影響實際運營業務的複雜性。  
一般來說，與 C Corp 相比，LLC 更容易成立，因為您只需在州政府註冊，提交組織章程，並在 LLC 的不同成員之間形成運營協議。  
一旦創建了 LLC，就可以通過基本的記錄保存來維護它，因為它不需要復雜的文書工作和治理。  
另一方面，當您向州政府註冊業務時，註冊為 C Corp 可能會帶來許多複雜性。

要註冊為 C Corp，您必須提交公司章程，然後您必須完成公司成立後的要求，包括發行創始人股票、填寫 83 (b)、創建董事會、創建章程等等。  
建立C Corp不僅會涉及更多複雜性，而且還會帶來更多成本。  
此外，維護 C 公司需要復雜的文書工作和治理，這增加了創建此類公司的複雜性。

在選擇 LLC 或 C Corp 進行籌款時，您應該考慮組建每種類型的公司所涉及的複雜程度以及相關成本。

### 責任保護

如果您正在考慮將 LLC 和 C Corp 作為您的業務結構，那麼您很可能希望為您的個人資產提供責任保護。  
LLC 和 C Corp 共同擁有的一件事是，它們都可以保護您的資產，以防企業產生責任。


## 為 LLC 與 C Corp 籌款

在選擇 LLC 或 C Corp 進行籌款時，一旦您考慮了所有其他重要因素，您就可以專注於您業務的籌款方面。  
但是，對於這兩種業務結構而言，確保投資看起來可能非常不同。  
雖然每個企業的資本要求可能不同，但獲得資本的能力可能取決於它是 C Corp 還是 LLC。  
因此，事不宜遲，以下是這兩種類型公司的籌款情況：

### 作為 C 公司籌款

在籌款方面，C Corp 可以勝過 LLC，原因很簡單，C Corp 可以發行其股票。  
C Corp 發行股票的能力意味著投資者發現投資這種公司結構很方便。  
投資者可以簡單地獲得 C Corp 的一定數量的股份，公司獲得運營和發展所需的資金。  
通過提供公司股票，C Corp 可以在短時間內籌集大量資金。  
除了提供股票以換取投資的能力外，C Corp 對投資者的風險也較小。

C Corps通過的雙重徵稅也是使他們對投資者有吸引力的一個因素。  
投資者很少願意投資通過所有者個人所得稅報告其損益的公司。  
他們更願意投資於一個報告其收入和損失的實體。  
如果非 C Corp 企業向投資者尋求投資，投資者可以要求該企業首先轉換為 C Corp，然後才能獲得投資。  
因此，簡而言之，成立 C Corp 會增加您獲得投資的機會，因為對於 C Corp 而言，籌款更簡單。

### 作為有限責任公司籌款

當有限責任公司試圖通過尋求投資者的投資來籌集資金時，他們面臨著一個根本性的挑戰，即缺乏股票可供發售。  
與 C 公司不同，有限責任公司不能簡單地提供股票以換取資本。  
現在，有限責任公司可能會向投資者提供股權以換取投資。  
但是，為了讓投資者獲得上述股權，他們必須成為 LLC 的成員。  
成為有限責任公司的成員涉及被添加到組織的文章中，大多數投資者不希望這樣。
 

當投資者想要投資有限責任公司時，還有另一個主要障礙。  
如上一節所述，如果投資者想投資有限責任公司，他們必須進行大量的盡職調查。  
因為，與 C Corp 不同，LLC 不能根據其股價估值。  
因此，投資者必須投入更多時間和研究，以確定他們投資的企業是否真正值得所有者聲稱的價值。

### 重點

最重要的是，如果您是一家擁有足夠資金來開展和經營業務的初創公司，那麼 LLC 可能是您的不錯選擇。  
但是，如果您正在尋求籌集資金併計劃尋求投資，那麼 C Corp 是一個更好的選擇。  
特別是如果您有發展公司或最終上市的宏偉計劃。  
因此，請花時間了解如何組建 LLC 或 C Corp 進行籌款工作。

## 如何將 LLC 轉變為 C Corp？

如果您是一家難以獲得投資的有限責任公司，那麼您可能希望將您的有限責任公司轉換為 C 公司。  
有兩種方法可以將 LLC 轉換為 C Corp，它們的工作方式如下：

### 法定轉換

法定轉換是將有限責任公司轉換為 C 公司的最簡單方法，它涉及在不解散公司的情況下將有限責任公司的資產和負債轉移給 C 公司。  
一旦法定轉換成功完成，有限責任公司的成員將成為新成立的 C Corp. 的股東。  

話雖如此，並非每個州都可以選擇法定轉換，因此您必須確保您的州允許這種轉換。

### 法定合併

如果由於特定州的限製而無法進行法定轉換，則可以使用法定合併將 LLC 轉換為 C Corp。
在法定合併期間，新的 C Corp 成立，LLC 的前成員成為新成立的公司的股東。
 
一旦 LLC 的成員成為新 C Corp 的股東，LLC 即解散，C Corp 可以開始運營。


## 結論

籌款一直是創業公司的主要考慮因素，如果沒有足夠的投資，在公司成立後的一段時間內很難支付公司的費用。
儘管有限責任公司更容易成立，但由於投資者面臨許多障礙，對於希望籌集資金的初創公司來說，它並不總是最好的選擇。
既然您了解了 LLC 或 C Corp 的籌款方式，您應該能夠在構建業務時做出更明智的決定。
